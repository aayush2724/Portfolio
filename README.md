# Aayush's Portfolio

> A modern, interactive portfolio built with React, Vite, and Tailwind CSS — showcasing projects, skills, and the journey of a 2nd year CS student.

## 🎯 Overview

This is my personal portfolio website featuring:

- **Modern Design**: Dark theme with glass-morphic UI and smooth animations
- **Optimized Cursor**: Custom crosshair cursor with glow effects
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile
- **Interactive Sections**: About, Skills, Projects, Life, Timeline, and Contact
- **Fast Performance**: Built with Vite for lightning-fast development and production

## 🛠️ Tech Stack

- **Frontend**: React 18 + Vite 5
- **Styling**: Tailwind CSS + PostCSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment Ready**: Optimized build for production

## 📋 Sections

- **Hero**: Eye-catching introduction with animated particles
- **About**: My story, background, and approach to learning
- **Skills**: Technical expertise in Full-Stack development, DSA, and more
- **Projects**: Showcase of real-world projects (RSB Secure, Skillnest, Disaster Relief)
- **Life**: Balance beyond code — Guitar, reading, and growth
- **Timeline**: Journey through key milestones
- **Contact**: Get in touch for collaborations and opportunities
- **Command Palette**: Quick navigation with `⌘K` / `Ctrl+K`

## 🚀 Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/aayush2724/Portfolio.git
cd Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

The portfolio will be available at `http://localhost:5173/`

## 📦 Build & Deploy

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

The optimized build will be in the `dist/` folder.
